# AI SQL Copilot

AI SQL Copilot converts business questions into SQL queries using an LLM.

## Features
- Natural language to SQL
- CSV upload
- Automatic schema inference
- SQL dialect selection
- SQL download
- Query history

## Tech Stack
- openai
- python-dotenv
- streamlit
- pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
